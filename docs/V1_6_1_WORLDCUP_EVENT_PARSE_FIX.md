# v1.6.1 世界杯赛事识别修正

## 修正重点

1. 修正世界杯战情室出现「未识别赛事」的问题。
2. 新增赛事类型判断：
   - 单场赛事：投注详情内含「A队 v B队」
   - 阶段盘口：例如「冠军盘口 玩法：入围16强」
   - 串关/多场：串关与 parlay 相关注单
3. 战情室的「最高流水赛事 / 平台盈利最高 / 平台亏损最高 / TOP10赛事」只统计单场赛事。
4. 阶段盘口不再归入「未识别赛事」，而是归为「阶段盘口：入围16强」等类别。
5. 清理投注详情中的 `_x000D_`，避免 Excel 换行符影响赛事解析。

## 数据规则

世界杯注单仍沿用：

```sql
`投注详情` LIKE '%世界杯2026(在加拿大、墨西哥&美国)%'
AND LOWER(`投注详情`) NOT LIKE '%panda%'
```

## 验收重点

进入：

```text
世界杯专区 → 战情室
```

确认以下位置不再出现「未识别赛事」：

- 最高流水
- 平台盈利最高
- 平台亏损最高
- TOP10 流水赛事
- TOP10 平台盈亏赛事

阶段盘口资料会保留在阶段汇总中，但不会污染单场比赛排行。
