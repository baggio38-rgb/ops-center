# v1.6.0 世界杯作战中心

本版继续扩展「亿兆智能决策平台」的世界杯专区。

## 新增内容

- 新增「赛程中心」：按小组赛、32强、16强、8强、半决赛、季军赛、冠军赛查看每场数据。
- 新增「比赛快速查看」：选择比赛后直接显示 KPI 与玩法分析。
- 新增「世界杯战情室」：显示最高流水、最高平台盈利、最高平台亏损、最高 RTP 赛事。
- 继续沿用 FIFA 2026 PNG 原图 Logo。
- 继续沿用深绿 + 金色世界杯主题。

## 世界杯注单识别规则

```sql
`投注详情` LIKE '%世界杯2026(在加拿大、墨西哥&美国)%'
AND LOWER(`投注详情`) NOT LIKE '%panda%'
```

## 验收项目

1. 世界杯专区能看到「赛程中心」。
2. 世界杯专区能看到「战情室」。
3. 赛程中心能按阶段显示比赛。
4. 选择比赛后能显示该场 KPI 与玩法分析。
5. 战情室能显示 TOP10 流水赛事与平台盈亏赛事。
