# 亿兆智能决策平台 Project Bible

## 项目定位

亿兆智能决策平台（YEIP）是一套企业级运营决策平台，用于整合投注、会员、财务、代理、风控与赛事数据，让管理层与各部门快速理解运营状态并做出决策。

## 核心原则

1. BigQuery 是唯一数据来源。
2. Dashboard 只负责展示，不负责业务计算。
3. 所有 KPI 必须可追溯到 Aggregate 表。
4. 新功能先设计数据模型，再开发页面。

## 核心模块

- 运营总览
- 世界杯专区 / 体育赛事中心
- 会员中心
- 风险中心
- 财务中心
- 代理中心
- 数据中心
- AI Copilot
